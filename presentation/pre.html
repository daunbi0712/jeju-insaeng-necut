React · Vite Canvas 2D Google Sheets + Apps Script GitHub → Render
Overview

브라우저만 열면 바로 찍는 인생네컷

설치도, 서버도 없이 — 웹캠 권한만 허용하면 4컷 촬영부터 다운로드까지 한 화면에서 끝납니다.

실시간 웹캠 프리뷰

getUserMedia로 카메라를 켜고 좌우 반전 프리뷰 제공

3·2·1 자동 촬영

카운트다운 후 자동 캡처를 4번 반복

제주 프레임 합성

돌하르방·야자수·감귤을 canvas로 실시간 합성

다운로드 / 재촬영

완성본을 PNG로 저장하거나 바로 다시 촬영

User Flow

촬영은 네 단계로 끝난다

STEP 01

카메라 켜기

페이지 진입 시 브라우저가 웹캠 권한을 요청

STEP 02

4컷 카운트다운

3·2·1 후 촬영, 총 4회 자동 반복

STEP 03

프레임 합성

4장을 제주 컨셉 프레임에 canvas로 합성

STEP 04

다운로드

완성 이미지를 저장하거나 다시 촬영

Frontend

React + Vite, 그리고 Canvas 하나로

React 18촬영 단계(대기·카운트다운·촬영·완료)를 상태로 관리
Vite개발 서버 · 프로덕션 번들링
getUserMedia브라우저 표준 API로 웹캠 스트림 획득
